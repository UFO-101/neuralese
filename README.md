## Getting started

1. Install [UV](https://docs.astral.sh/uv/)

```bash
wget -qO- https://astral.sh/uv/install.sh | sh
```

2. Install the dependencies:

```bash
uv sync
```

3. Install the pre-commit hooks:

```bash
pre-commit install
```
